How Q&A Pairs Become Training Strings

Models don't read JSON. They read one long string. Here's the conversion.

Step 1: You Write Q&A as JSON (Easy for Humans)

{
  "instruction": "What is Metformin?",
  "input": "",
  "output": "Metformin is a biguanide used to treat type 2 diabetes."
}

This is how YOU write it. Clean. Structured. Three fields.

↓

Step 2: The Model Sees ONE String (Easy for Machines)

Below is an instruction. Write a response.

### Instruction:
What is Metformin?

### Response:
Metformin is a biguanide used to treat type 2 diabetes.

This is what the MODEL reads. One big string. The ### markers separate the question from the answer.

Step 3: During Training — The Model Predicts the Answer

✅ What the Model SEES (Input)

Below is an instruction...

### Instruction:
What is Metformin?

### Response:

Everything BEFORE the answer

🎯 What It Must PREDICT (Label)

Metformin is a biguanide used to treat type 2 diabetes.

ONLY the answer part

🧠 The ### markers are the secret. During training, the model learns: "When I see '### Response:', whatever comes next is what I should generate." After training, you give it a new instruction ending with '### Response:' — and it auto-completes the answer.

Why Not Just Feed JSON to the Model?

❌ If you fed JSON

{"instruction": "What is Metformin?", "output": "Metformin is..."}

Model would learn to generate... more JSON. Not answers. It would become a "JSON generator" not a "question answerer."

✅ With Alpaca template

### Instruction: What is Metformin? ### Response: Metformin is...

Model learns to generate the ANSWER. The template tells it "question ends here, answer starts here."

Your 5 Q&A Pairs → 5 Training Strings

Each one becomes its own training example. Same pattern, different content.

Example 1: ### Instruction: What is Metformin? ### Response: Metformin is a biguanide... Example 2: ### Instruction: How does Metformin work? ### Response: It decreases hepatic glucose... Example 3: ### Instruction: What are common side effects? ### Response: Common side effects include nausea... ...and so on for all 5 examples.

After training on these 5 strings, the model learns: "When someone asks about Metformin in the ### Instruction: format, I should answer in the ### Response: format."
