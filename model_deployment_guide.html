Deploy Your Fine-Tuned Model

From Colab to production — 4 levels, from local testing to public API

You already have: ./medical_domain_adapter/ with adapter_model.safetensors (4.5 MB) + tokenizer.json + adapter_config.json
Level 1

🔧 Load Adapter — Use Locally

Load base model + adapter separately. No merging needed. Good for testing.

# Load base model + your trained adapter from transformers import AutoModelForCausalLM, AutoTokenizer, pipeline from peft import PeftModel base = AutoModelForCausalLM.from_pretrained( "TinyLlama/TinyLlama-1.1B-Chat-v1.0", device_map="auto" ) model = PeftModel.from_pretrained(base, "./medical_domain_adapter") tokenizer = AutoTokenizer.from_pretrained("./medical_domain_adapter") pipe = pipeline("text-generation", model=model, tokenizer=tokenizer) pipe("Metformin is used to treat")
Level 2

🧬 Merge — One Normal Model (Phase 3.10!)

Bake B×A INTO the base weights. Adapter disappears. No LoRA needed at inference.

# Merge adapter INTO base weights (one-time operation) model = model.merge_and_unload() model.save_pretrained("./merged_medical_model") tokenizer.save_pretrained("./merged_medical_model") # Now use it WITHOUT peft — it's a normal transformers model! model = AutoModelForCausalLM.from_pretrained("./merged_medical_model") pipe = pipeline("text-generation", model=model, tokenizer=tokenizer) # No PeftModel. No adapter_config. Just a normal model.
Level 3

🌐 FastAPI — Turn Into an API

Create a REST API your app/website can call. 10 lines of code.

# File: app.py from fastapi import FastAPI from pydantic import BaseModel from transformers import pipeline app = FastAPI() pipe = pipeline("text-generation", model="./merged_medical_model") class Request(BaseModel): prompt: str @app.post("/generate") def generate(req: Request): result = pipe(req.prompt, max_new_tokens=100) return {"output": result[0]["generated_text"]} # Run: uvicorn app:app --reload # API at: http://localhost:8000/generate # Test: curl -X POST http://localhost:8000/generate -d '{"prompt":"Hello"}'
Level 4

🚀 Push to HuggingFace — Share with the World

Upload to HF Hub. Anyone can download. Free inference API included.

# Login (one-time) from huggingface_hub import login login() # Enter your HF token (from huggingface.co/settings/tokens) # Push merged model model.push_to_hub("ashokmulchandani/tinyllama-medical") tokenizer.push_to_hub("ashokmulchandani/tinyllama-medical") # Now ANYONE can use it: # model = AutoModelForCausalLM.from_pretrained("ashokmulchandani/tinyllama-medical") # Or via free HF API: https://api-inference.huggingface.co/models/ashokmulchandani/tinyllama-medical
