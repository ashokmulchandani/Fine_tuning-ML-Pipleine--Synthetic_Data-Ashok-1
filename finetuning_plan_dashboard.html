🔢 Quantization Scale Factor

scale = (x_max − x_min) / (q_max − q_min) quantized_value = round(original_value / scale) FP32 → FP16 → INT8 → INT4 (higher memory → lower memory)

🧬 LoRA (Low-Rank Adaptation)

W_new = W_0 + B × A where: W_0 = frozen pre-trained weights (d × d) B = trainable matrix (d × r) A = trainable matrix (r × d) r = rank (typically 8) 175B model with r=8 → only 37.7M trainable params (0.02%)

🎯 DPO Loss (Direct Preference Optimization)

L_DPO = −E[ log σ( β × ( log π_θ(y_w|x)/π_ref(y_w|x) − log π_θ(y_l|x)/π_ref(y_l|x) ) ) ] where: π_θ = policy model (being trained) π_ref = reference model (frozen instruction model) y_w = chosen/winning response y_l = rejected/losing response β = temperature (0.1–0.5)
